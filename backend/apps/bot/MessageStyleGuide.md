# Message Style Guide: Attila Recruiting Telegram Bot

## Цель
Создать единообразный, премиум-стиль сообщений, который снижает когнитивную нагрузку и повышает конверсию в ключевые действия.

---

## Общие принципы

### 1. Tone of Voice
- **Дружелюбно, но профессионально** - как опытный HR, который на твоей стороне
- **Уважительно** - обращение "вы" (кроме неформальных каналов по согласованию)
- **Конкретно** - никаких "возможно", "может быть", "постараемся"
- **Мотивирующе** - каждое сообщение = маленький шаг к цели

### 2. Структура сообщения

**Оптимальная структура:**
```
[Заголовок с эмодзи]
[Краткое описание контекста]

[Ключевая информация блоками]

[Чек-лист или действия (если нужно)]

[Призыв к действию]
```

**Правило "3-4 блоков":** Сообщение не должно быть "простынёй". Максимум 3-4 логических блока, разделённых пустой строкой.

### 3. Длина строк
- **Макс 60 символов на строку** (удобно на мобильном)
- Длинные предложения разбивать на несколько строк
- Один логический блок = один параграф

## Эмодзи-маркеры

Используем эмодзи для визуального разделения типов информации:

| Эмодзи | Значение | Пример использования |
|--------|----------|----------------------|
| 📅 | Дата | 📅 Пн, 12 дек |
| 🕐 / ⏰ | Время / Напоминание | ⏰ 14:30 (МСК) |
| 📍 | Место / Адрес | 📍 ул. Ленина, 10, офис 5 |
| 💬 | Формат встречи | 💬 Видеозвонок • 15-20 мин |
| 🔗 | Ссылка | 🔗 Яндекс.Телемост: https://... |
| ✨ | Важная информация | ✨ Подготовьтесь заранее |
| ⚡ | Действие / Активность | ⚡ Что взять с собой |
| 🎯 | Цель / Результат | 🎯 Следующий шаг |
| ✅ | Подтверждение / Успех | ✅ Встреча подтверждена |
| 🔁 | Перенос / Изменение | 🔁 Выберите новое время |
| 💼 | Контакт / Рекрутер | 💼 Иванов Иван: +7 900 123-45-67 |
| 👋 | Приветствие | 👋 Добрый день, Анна! |
| 🎉 | Поздравление | 🎉 Поздравляем! |

**Правило использования:**
- Максимум 1 эмодзи на строку (в начале)
- Не ставим эмодзи в середине предложения
- Эмодзи заменяет маркер списка (вместо • или -)

## Форматирование даты и времени

### Полный формат (с днём недели)
```
Пн, 12 дек • 14:30 (МСК)
```

**Правила:**
- День недели сокращённо: Пн, Вт, Ср, Чт, Пт, Сб, Вс
- Месяц сокращённо: янв, фев, мар, апр, мая, июн, июл, авг, сен, окт, ноя, дек
- Разделитель даты и времени: `•` (средняя точка)
- Часовой пояс в скобках: (МСК), (НСК), (по вашему времени)

### Короткий формат (без дня недели)
```
12.12 • 14:30
```
Используется в кратких напоминаниях или списках.

### Относительные даты
- **Сегодня:** "сегодня в 14:30"
- **Завтра:** "завтра в 14:30"
- **Послезавтра:** "13 дек в 14:30" (уже полная дата)

### Диапазоны времени
```
14:30 - 15:00 (30 мин)
```

## HTML-разметка

Telegram поддерживает базовый HTML. Используем:

### Поддерживаемые теги:
- `<b>жирный</b>` - для заголовков и важных слов
- `<i>курсив</i>` - для подсказок и уточнений
- `<code>код</code>` - для технических деталей (редко)
- `<a href="url">ссылка</a>` - для кликабельных URL

**НЕ используем:**
- `<u>` (подчёркивание) - плохо читается
- `<s>` (зачёркивание) - не используем
- Вложенные теги (проблемы с парсингом)

### Примеры:
✅ **Правильно:**
```html
<b>✅ Встреча подтверждена</b>

📅 Пн, 12 дек • 14:30 (МСК)
💬 Видеозвонок • 15-20 мин
```

❌ **Неправильно:**
```html
<b><i>Встреча подтверждена</i></b> ✅  <!-- вложенность -->
📅 <b>Пн</b>, 12 дек • 14:30 (МСК)  <!-- эмодзи внутри тегов -->
```

## Чек-листы

Используем для подготовки к встречам:

```
⚡ <b>Подготовьтесь заранее:</b>

✓ Стабильный интернет (минимум 5 Мбит/с)
✓ Тихое место для разговора
✓ Наушники с микрофоном (опционально)
✓ 2-3 вопроса о вакансии
```

**Правила:**
- Символ ✓ или □ в начале каждого пункта
- Каждый пункт = конкретное действие
- Максимум 5 пунктов (иначе перегрузка)

---

## Кнопки (Inline Keyboard)

### Стиль текста кнопок:
- **Глаголы в повелительном наклонении:** "Подтвердить", "Выбрать время", "Отменить"
- **Короткие:** максимум 20 символов
- **Без эмодзи в кнопках** (эмодзи только в тексте сообщения)

### Примеры кнопок:
```
✅ Подтвердить участие | ❌ Не смогу прийти
```
```
📅 Выбрать другое время
```

**Правила расположения:**
- 1-2 основных действия в первом ряду
- Второстепенные действия ("Отменить", "Назад") - во втором ряду
- Максимум 2 кнопки в ряду (на мобильном удобнее)

## Сценарии сообщений

### 1. Подтверждение записи
**Цель:** Информировать + мотивировать готовиться

```html
<b>✅ Встреча подтверждена</b>

👋 {candidate_name}, вы шаг ближе к команде SMART!

📅 {datetime_local}
💬 Видеозвонок • 15-20 мин

⚡ <b>Подготовьтесь заранее:</b>

✓ Стабильный интернет
✓ Тихое место для разговора
✓ 2-3 вопроса о вакансии

🔔 Поставьте напоминание на телефон.
Ссылка придёт за 2 часа до встречи.
```

### 2. Напоминание за 6 часов
**Цель:** Подтвердить намерение (снизить no-show)

```html
<b>⏰ Встреча сегодня</b>

📅 {datetime_local}

Подтвердите участие, чтобы мы подготовили
всё необходимое.
```
[Кнопки: "Подтвердить участие" | "Не смогу"]

### 3. Напоминание за 2 часа + ссылка
**Цель:** Дать ссылку + последний reminder

```html
<b>⏰ Встреча через 2 часа</b>

📅 {datetime_local}

🔗 <b>Ссылка на встречу:</b>
{meet_link}

Если планы изменились — сообщите,
и мы подберём другое время.
```

### 4. Инструкция перед созвоном
**Цель:** Снизить стресс, дать чек-лист

```html
<b>✨ Готовимся к встрече</b>

📅 {datetime_local}
🔗 {meet_link}

⚡ <b>Чек-лист перед созвоном:</b>

✓ Проверьте интернет (speedtest.net)
✓ Закройте лишние вкладки/приложения
✓ Приготовьте воду
✓ Улыбнитесь — это не экзамен! 😊

До встречи через {time_left}!
```

### 5. Итог созвона + запись на ОД
**Цель:** Поздравить, дать next step

```html
<b>🎉 Поздравляем!</b>

Вы успешно прошли видеоинтервью в компанию SMART!

🎯 <b>Следующий шаг:</b>
Ознакомительный день (2 часа)

📅 Выберите удобное время:
```
[Кнопка: "Выбрать дату"]

### 6. Инструкция на ОД
**Цель:** Дать всю инфо, мотивировать

```html
<b>🎉 Ознакомительный день</b>

📅 {datetime_local}
📍 {address}

✨ <b>Что это?</b>
Это не собеседование, а знакомство с командой
и форматом работы (~2 часа).

⚡ <b>Что взять с собой:</b>

✓ Паспорт
✓ Блокнот и ручка (опционально)
✓ Отличное настроение

💼 <b>Контакт руководителя:</b>
{contact_name}: {contact_phone}

<i>Приходите в презентабельном виде.
Проявите активность — это важно!</i>
```

### 7. Перенос/отмена
**Цель:** Бережно предложить альтернативу

```html
<b>🔁 Изменение встречи</b>

Мы освободили бронирование на {old_datetime}.

Давайте подберём другое время, которое вам удобно:
```
[Кнопка: "Выбрать новое время"]

### 8. "Не дозвонились" (gentle)
**Цель:** Не обидеть, дать второй шанс

```html
<b>📞 Не удалось связаться</b>

Мы пытались дозвониться {datetime_local},
но не смогли.

Если у вас всё ещё есть интерес —
выберите новое удобное время:
```
[Кнопка: "Выбрать время"]

## Запрещённые практики

❌ **Не делаем:**
- Длинные параграфы без разбивки (>4 строк)
- Несколько эмодзи подряд (🎉🎉🎉)
- КАПС для привлечения внимания (используем `<b>`)
- Неопределённость: "возможно", "постараемся", "попробуем"
- Формальный канцелярит: "в связи с вышеизложенным"
- Избыточные извинения: "извините за беспокойство"
- Хардкод конкретных дат/имён (только через переменные)

✅ **Делаем:**
- Короткие предложения (макс 2 строки)
- Один эмодзи = один смысловой блок
- Жирный текст для заголовков
- Конкретность: "встреча 12 декабря в 14:30"
- Простой язык: "как у нас принято" вместо "в соответствии с регламентом"
- Позитивный тон: "отлично!" вместо "приемлемо"
- Переменные: {candidate_name}, {datetime_local}

## Переменные в шаблонах

### Обязательные переменные:
```jinja
{{ candidate_name }}         # Имя кандидата
{{ datetime_local }}         # Дата/время в локальной TZ
{{ city_name }}              # Название города
{{ recruiter_name }}         # Имя рекрутера
{{ meet_link }}              # Ссылка на встречу
{{ address }}                # Физический адрес
{{ contact_phone }}          # Телефон контакта
```

### Фильтры Jinja2:
```jinja
{{ start_utc|format_datetime(tz) }}    # Полный формат: Пн, 12 дек • 14:30 (МСК)
{{ start_utc|format_date(tz) }}        # Только дата: Пн, 12 дек
{{ start_utc|format_time(tz) }}        # Только время: 14:30 (МСК)
{{ duration_minutes }}                 # 30, 60 и т.д.
```

## Checklist для каждого нового шаблона

Перед релизом проверяем:

- [ ] Длина строк ≤ 60 символов
- [ ] Использованы эмодзи-маркеры (но не избыточно)
- [ ] HTML теги закрыты корректно
- [ ] Нет вложенных тегов `<b><i>`
- [ ] Единый формат даты/времени
- [ ] Максимум 3-4 блока в сообщении
- [ ] Есть призыв к действию (или кнопка)
- [ ] Все переменные используют шаблонизацию (не хардкод)
- [ ] Tone of voice: дружелюбно, но профессионально
- [ ] Нет запрещённых практик (КАПС, канцелярит, etc.)

## Примеры до/после

```
Здравствуйте! Напоминаем, что у вас запланирована встреча с рекрутером компании SMART SERVICE на 12.12.2024 в 14:30 по московскому времени. Пожалуйста, подтвердите ваше участие, нажав на кнопку ниже. В случае, если вы не сможете присутствовать, просим вас сообщить об этом заранее. Спасибо за понимание.
```

```html
<b>⏰ Встреча сегодня</b>

📅 Пн, 12 дек • 14:30 (МСК)

Подтвердите участие, чтобы мы подготовили

```

**Улучшения:** Короткая структура, эмодзи, единый формат даты, чёткий CTA.
